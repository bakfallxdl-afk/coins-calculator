# 🪙 Coins Calculator PWA

一个跨平台的硬币分配计算器，支持响应式设计和离线使用。

## ✨ 功能特性
- 🎯 智能硬币分配算法
- 📱 响应式设计（适配手机和电脑）
- 👥 支持4-6名玩家
- 🎲 随机Roll点功能
- 📊 道具扣减处理
- 🌐 PWA支持（可添加到主屏幕）
- 🇨🇳 中英文双语界面
- 📁 使用GitHub Desktop管理项目

## 🚀 快速开始
